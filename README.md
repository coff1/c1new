# c1new    
一个信息收集/网站爬虫脚本  
目前支持：  
  公司备案域名  
  端口扫描  
  子域名收集  
  存活检测  
  站点铭感信息爬虫  
用法
1，python c1new.py -t target
target 可以是string 可以是txt文件，支持ip，域名，公司名，url

2，python 从new.py -s all
显示任务id

3，python c1new.py -i 任务id -o 文件名.xlsx
输出结果
